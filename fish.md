# fish shell

<https://fishshell.com/docs/current/>

## `bind`

fish equivalent to `bindkey`

## shortcuts

- `<A-p>` - pipe command to `less`
- `<C-g>` - pipe command to `less`
- `<C-;>` - pipe command to `less`
- `<A-r>` - pipe command to `less`
